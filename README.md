# 就活スクールサイト（学習用）

Hello Mentor の教材をもとに、Web 制作の基礎力向上を目的として制作しました。

## 概要

このリポジトリは学習用として作成した就活スクールサイトです。

## 使用技術

- HTML5
- SCSS
- CSS3
- JavaScript

## 工夫した点

- 実務を想定し、無駄な試行錯誤を減らすことでコーディングスピードを意識しました。
- JavaScriptによるアニメーションを取り入れ、過度にならない動きを意識して実装しました。
- デザインカンプの意図を汲み取り、忠実に再現しました
- 後から見返したときに分かりやすいよう、クラス名の付け方を意識しました
